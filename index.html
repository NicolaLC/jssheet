Create Fast - Reusable css stylesheets!

What is?

It's a new way to write css using Javascript!

How does it works?

Starting from a :root element you can define css properties and childrens: 

      
const stylesheets = {
  root: {
    selector: "body",
    background: colors.primary,
    fontFamily: fonts.main,
    padding: 0,
    margin: 0,
    overflowX: "hidden",
    childrens: [header, body, ...generics]
  }
};
